# 🚀 PrestaShop 8 Docker Boilerplate para Desarrollo de Módulos

Este entorno está optimizado para desarrollar módulos de PrestaShop de forma aislada y rápida usando Docker y MariaDB.

## 🛠️ Cómo usarlo
1. Clona este repositorio.
2. Copia el archivo `.env.example` y renómbralo a `.env`.
3. Crea tu código dentro de la carpeta `/src_modulo`.
4. Ejecuta `docker compose up -d`.
5. Tu entorno limpio estará en `http://localhost:8080`.
